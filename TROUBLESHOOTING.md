# Treasure Hunt Solver - Troubleshooting Guide

## Lỗi "Failed to solve treasure hunt"

### 1. Kiểm tra Backend có đang chạy không

**Cách 1: Sử dụng script debug**
```bash
cd TreasureHuntApp
.\debug-backend.bat
```

**Cách 2: Khởi động thủ công**
```bash
cd TreasureHuntApp/Backend
dotnet run
```
